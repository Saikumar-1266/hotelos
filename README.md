# HotelOS

HotelOS is a modern hotel management platform designed for hotels that provide restaurant services, food ordering, room stays, and banquet hall bookings.

## Core Features

- Restaurant and food ordering
- Tea and tiffin menu
- Room reservations
- Banquet hall bookings
- Hotel owner dashboard

## Tech Stack

- React
- Supabase
- Vercel
- GitHub
